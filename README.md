# Enhanced-Skywars-Lite
強化空島戰爭 Lite，是2019年的地圖「[強化空島戰爭](https://forum.gamer.com.tw/C.php?bsn=18673&snA=180283)」的續作。本作是一張空島型PVP地圖，玩家們會分成2 ~ 4隊，目標是利用諸多島嶼內的資源，生存下去並殲滅對方。
